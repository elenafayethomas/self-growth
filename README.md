## Usage

Can be served as a local website, at http://localhost:3000/docs/default/component/local/, using [techdocs](https://backstage.io/docs/features/techdocs/cli/).

Install:

```shell
npm i -g @techdocs/cli
```

Run:

```shell
techdocs-cli serve --no-docker
```
